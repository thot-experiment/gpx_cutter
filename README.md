# GPX Cutter

GPX Cutter is a simple, browser-based tool that allows you to visualize a GPX track and extract a specific section of it. It is designed to help you remove irrelevant parts of a GPS log (like the commute to and from a trail) while keeping the relevant activity.

## Features

- **Easy Upload**: Drag and drop your `.gpx` file directly into the browser.
- **Path Visualization**: View your track on a 2D canvas, color-coded by speed (Blue $\rightarrow$ Green $\rightarrow$ Yellow $\rightarrow$ Red).
- **Metric Analysis**: Analyze your track with a dynamic graph.
  - **Y-Axis**: Toggle between **Speed** and **Elevation**.
  - **X-Axis**: Toggle between **Time** and **Distance**.
- **Interactive Cutting**: Drag and select a region on the graph to define the section you want to keep.
- **GPX Export**: Download the selected segment as a new `.gpx` file, preserving the original XML structure.
- **Real-time Tooltips**: Hover over the path or graph to see precise speed, distance, and elevation data.

## How to Use

1. Open `index.html` in any modern web browser.
2. Drag and drop your GPX file or click **Select File**.
3. (Optional) Use the graph controls to switch between Speed/Elevation and Time/Distance to find the exact section you need.
4. **Click and drag** on the graph to highlight the portion of the track you wish to keep.
5. Click **Download Selected Section** to save the clipped GPX file to your computer.

## Technical Details

- **Frontend**: Built with vanilla HTML, CSS, and JavaScript (ES Modules).
- **Rendering**: Uses the HTML5 Canvas API for high-performance path and graph rendering.
- **Calculations**: 
  - Uses the **Haversine formula** to calculate distances between GPS coordinates.
  - Implements a moving average smoothing algorithm for cleaner metric graphs.
- **Processing**: GPX files are processed entirely in the browser using `DOMParser` and `XMLSerializer`, ensuring your data never leaves your local machine.
